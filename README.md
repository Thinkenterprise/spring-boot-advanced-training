# spring-boot-advanced-training
Spring Boot Advanced Training


## Build 

### Update Maven & Gradle Wrapper 


gradle wrapper --gradle-version 6.5.1

mvn -N io.takari:maven:0.7.7:wrapper -Dmaven=3.6.3


### Maven Build 

./mvnw clean package

./mvnw  spring-boot:start

### Maven Build 
./gradlew build

./gradlew bootRun
